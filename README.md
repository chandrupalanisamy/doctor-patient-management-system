# Doctor Patient Management System

A web-based Doctor Patient Management System developed using PHP and MySQL.

## Features
- Patient Registration
- Doctor Login
- Appointment Booking
- Admin Dashboard
- Patient Records Management
- Secure Authentication

## Technologies Used
- PHP
- MySQL
- HTML
- CSS
- JavaScript
- Bootstrap

## How to Run
1. Install WAMP/XAMPP
2. Import the SQL database
3. Place project folder inside `www` or `htdocs`
4. Start Apache and MySQL
5. Open localhost in browser
